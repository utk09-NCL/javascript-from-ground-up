# JavaScript from Ground Up

This repository contains a comprehensive guide and resources for learning JavaScript from basics.

**"Whenever there's a divergence between what your brain thinks is happening, and what the computer does, that's where bugs enter the code."** - Kyle Simpson

## How to use this repository

Each markdown file corresponds to a module in the JavaScript from Ground Up series. You can read through the files sequentially to build your understanding of JavaScript concepts.

- [Module 1: Foundations and Types](01_foundations_and_types.md)
- [Module 2: Data Structures](02_data_structures.md)
- [Module 3: Coercion and Equality](03_coercion_and_equality.md)
- [Module 4: Scope, Hoisting, and Closures](04_scope_hoisting_and_closures.md)
- [Module 5: Objects in Depth](05_objects_in_depth.md)
- [Module 6: Prototypes and Classes](06_prototypes_and_classes.md) (Upcoming)

There's also a [`ghw_event_system.js`](ghw_event_system.js) file with tasks and solutions for building an event registration system. Try solving each task before looking at the solution!
